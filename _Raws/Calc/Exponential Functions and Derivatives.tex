\section{Exponential Functions}

A function in the form
\[
	y = C^{x}
\]
example:
\[
	y = 5^{x}
\]

Where:

\begin{gather*}
	f(x) = 5^{x} \\
	g(x) = x+1\\[10pt]
	f[g(x)] = 5^{x+1}
\end{gather*}

The derivative of such a function:

\[
	y^{\prime} = C^{x}  \times x^{\prime}  \times  \ln(C)
\]

\ex{$y = 5^{-x}$}{
\[
	y^{\prime} = -5^{-x} \ln(5)
\]
}

\ex{$y = 5^{-2x}$}{
\[
	y^{\prime} = -10^{-2x} \ln(5)
\]
}
